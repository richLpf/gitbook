# 命令总览

Git命令很多，这里列举下：

## 配置

> 通过该命令可以配置Git信息，然后信息可以再后面显示

git config


## 创建仓库

> 通过这两个命令可以创建Git仓库

git init

git clone


## 基本操作

> 通过这几个命令可以提交文件，将文件提交到Git仓库，工作中基本这几个命令就够用了

git add

git commit 

git push


## 分支与合并

> Git创建分支，检出分支

git branch

git checkout 


## 其他

git diff

git log

git reset

git rm
