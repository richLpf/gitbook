# GitBook 博客搭建

### 一、生成项目

1、gitbook-cli安装

```
npm i -g gitbook-cli

gitbook -V
```

2、创建项目

```
mkdir git-learn

cd git-learn
```

3、打包构建

``` 
//默认生成_book下

gitbook build

gitbook build ./ ./docs
```

4、生成目录

```
gitbook init

gitbook serve

```

__目录结构见SUMMARY.md__
