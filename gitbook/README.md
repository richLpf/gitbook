# gitbook的使用

通过Github、GitBook、Git Actions 和 Typora 打造自己的博客系统

截图展示
## gitbook-cli安装

```
npm i -g gitbook-cli

gitbook -V
```
## 创建项目

```
mkdir git-learn

cd git-learn
```

## 打包构建

gitbook build 默认生成_book下

gitbook build ./ ./docs

## 生成目录

```
gitbook init

gitbook serve

```
__目录结构见SUMMARY.md__
