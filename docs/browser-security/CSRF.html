1.1. CSRF攻击

1.1.1. 一、前言

我们常常听到这样一句话：默认的链接不要点，那些年也听过，邮箱中的垃圾链接不要点。

因为可能是黑客发起的CSRF攻击，所以再点击之前最好是确认链接的安全性。

1.1.2. 二、什么是CSRF攻击

通过一个例子，说明CSRF攻击，早些年GMail邮箱有个漏洞，当然黑客发送攻击链接到邮箱后，用户登录邮箱打开
