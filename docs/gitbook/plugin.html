1.1. 安装插件

// book.json文件，见根目录
touch book.json

// 下载插件使用gitbook install,不要使用npm,会出现插件不起作用的情况,可能和网络有关
gitbook install

// 安装插件（不推荐使用）
npm install gitbook-plugin-[插件名]
